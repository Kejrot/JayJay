Belajar Dasar-Dasar HTML

HTML (HyperText Markup Language) adalah fondasi dari semua halaman web. Dengan memahami HTML, Anda dapat membuat struktur halaman web yang rapi dan terorganisir.

Pengenalan HTML

HTML terdiri dari berbagai elemen seperti heading, paragraf, link, dan daftar. Elemen-elemen ini digunakan untuk menyusun konten dalam halaman web.

Contoh Penggunaan Link

  • Bookmark link ke bagian daftar
  • Link ke W3Schools (dengan target=_blank)
  • Link ke gambar lokal (Image Link)
  • Internal Link ke Halaman 2
